Editing the configuration file

 The configuration file is in "XML". They look familiar to the HTML codes used in web pages. An introduction to this file format is available if you'd like to read more about it, but it's pretty self-explanatory. Your favorite editor program should be able to work with these files, as they are just text.

The files are stored in different places on different types of computers. It might be fastest to just search for the name you gave the file when you saved it. The usual locations for preference information (if you don't change directory when saving) is:

  • Linux - in ~/.jmri/
  • Mac OS X - in the Library/Preferences/JMRI directory in your home directory
  • Macintosh - in the JMRI folder within the Preferences folder within the System Folder
  • Windows - in C:\Program Files\JMRI, or MyDocuments\JMRI

Opening a panel at program startup

 The CornwallRR program was modified to automatically open the dispatcher panel at startup. This was an OK solution because that was a custom program anyway. But it's also possible to edit the configuration file so that a panel is automatically opened.
