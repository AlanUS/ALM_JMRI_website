JMRI CodeFlower

Introduction

The CodeFlower animation gives a lively perspective of how JMRI was built up and how incredibly diverse the current state of work is.

CodeFlower Source code visualization

CodeFlower visualizes source repositories using an interactive tree. Each disc represents a file, with a radius proportional to the number of lines of code (loc). All rendering is done client-side, in JavaScript. Try hovering on nodes to see the loc number, clicking on directory nodes to fold them and dragging nodes to rearrange the layout. Built with d3.js.

Enjoy!

Also view our 2009 CodeSwarm.

Technical Details

This CodeFlower was created using one local GitHub repository, cloc and functions embedded in the CodeFlower page to convert the information into a json format.

CodeFlower was built with d3.js, inspired by Code Swarm and Gource. It was published as open-source by François Zaninotto under the MIT license.
