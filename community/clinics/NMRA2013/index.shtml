JMRI: NMRA 2013 Clinics

4 JMRI clinics presented at 2013 NMRA convention in Atlanta.

Dick Bronson presented 2 clinics:

    Control Panel Editor Create a modern dispatch panel using Control Panel Editor
    Automatic train control using Warrants JMRI Warrants allow you to automatically run trains on panels created with the Control Panel Editor and using Signal Masts for Aspect Based Signaling.

Steve Todd presented a clinic on:

    "Using Phones and Pads for Throttles and Panels (JMRI)"

John Stewart and Craig Gardner did an Operations clinic:

    "JMRI Operations on a 4'X6' Layout." Using a small layout as a test bed you will see how to actually implement JMRI "Live". The idea was to show how easy it can be to "just do it".

More information on the convention can be found on the convention website.
