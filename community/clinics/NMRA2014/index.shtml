JMRI: NMRA 2014 Clinics

There will be several JMRI clinics at 2014 NMRA convention in Cleveland OH, including both old favorites and several new ones.

The current schedule has:

  • John Stewart will present three clinics on using JMRI Operations. Two of them are general but the third will focus on using Operations within a steel mill layout.
  • Ken Cameron will host a overview of JMRI. It is intended to show some of the structure and then expose all the major parts of JMRI so users are aware of features of JMRI they might not have been using before. Also he will highlight any new areas and latest changes in JMRI.
  • Dick Bronson will provide a clinic focusing on the Control Panel Editor and the features that this tool is optimal for using.
  • David Duchamp will present the features of using the Layout Editor. This is the simplest way to setup lots of information about your layout into JMRI so it can help with signals, controls and automation options.
  • Dick Bronson will show how to use the Warrants system to automate trains on your layout.
  • Ken Cameron and other developers will host the JMRI User Group meeting. Come and meet the folks behind the code and help shape the direction of JMRI.
  • Dick Johannes will present what the NER-HUB division is doing for modular layouts adding signals via the CATS extension to JMRI.

More information on the convention can be found on the convention website.
